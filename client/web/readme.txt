Steps to run sample web client:


pre-requisite:
1. Backend server should be run on http://localhost:8080
2. Internet should work properly.

Steps to run:
1. Open index.html in chrome
2. You will get log in right side which include user id, file location, and score
3. In left side, you need to choose measures
4. Allow permission to record audio in browser
5. Once recording started, Say Banana or Pa-Ta-Ka repeatedly
6. After recording, score will calculate and displayed in log.
7. Click on check again for score analysis for same user id otherwise reload the page from browser for new user
